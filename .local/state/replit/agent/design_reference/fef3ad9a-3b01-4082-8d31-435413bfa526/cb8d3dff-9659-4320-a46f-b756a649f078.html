Quick Start Guide

Get up and running with InitraJS in less than 2 minutes.

1

Install CLI

npm install -g initrajs

Install the InitraJS CLI globally on your system.

2

Create Project

initrajs create my-app

Scaffold a new project with your preferred template.

3

Start Building

cd my-app && npm start

Jump straight into development with zero configuration.
